# l4d2_rank_system

求生之路2积分排名插件，需要MySQL数据库

使用方法：创建一个MySQL数据库，配置databases文件后插件会自行创建数据表

指令:

!rank自己的积分，开关头衔

!rankall查看房间内所有人的积分

!ranktop查看全服前十

起因就是在网上没有找到满意的类似rank的插件，于是尝试用AI写了一个

所有源码由AI编写，所以遇到什么bug请自行尝试修复

根据我目前的测试正常使用应该不会有什么大问题

目前已知bug：

近战造成的友伤会使扣分异常

击倒冲锋的charger不加分

spitter累计伤害不加分

如果你会写插件修复这些应该不难

欢迎所有人自行修改源码或提交修复

需要注意如果代码被重新编译后加载插件可能会覆盖原有数据表
